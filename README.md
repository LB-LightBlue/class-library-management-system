# class-library-management-system
## 项目概述
本项目是 “图书角” 简易 Web 应用的初步设计，核心目标是实现图书信息管理、用户借阅 / 归还流程，为班级同学提供便捷的图书共享服务。本次作业完成了系统建模（用例图、类图）和项目版本控制初始化。
## UML 图设计说明
`用例图`：明确区分 “用户” 和 “管理员” 两类角色的核心操作，覆盖图书搜索、借阅、归还、信息录入、维护等核心功能，界定系统边界。<br>
`类图`：聚焦 “图书”“用户”“借阅记录” 三个核心实体，定义关键属性和方法，通过一对多关联体现实体间的依赖关系，为后续开发提供数据模型基础。
